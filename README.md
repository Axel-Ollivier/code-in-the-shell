# CITS Themes

**Code in the Shell** — a curated collection of VS Code color themes drawn from cinema, anime, and cult film. Each theme is built around a specific palette, a specific mood, a specific world.

Themes inspired by: Ghost in the Shell, Princess Mononoke, Spirited Away, Blade Runner 2049, Star Wars, Studio Ghibli, The Colour Out of Space, The Toxic Avenger, Hannah Montana, Dune, Cowboy Bebop, Annihilation, Arrival, Her.
