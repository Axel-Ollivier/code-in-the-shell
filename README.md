# CITS Themes

**Code in the Shell** — a curated collection of VS Code color themes drawn from cinema, anime, and cult film. Each theme is built around a specific palette, a specific mood, a specific world.

---

## Themes

- **Code in the Shell** `dark` — *Ghost in the Shell — Oshii, 1995*
- **Code in the Shell — Day** `light` — *Ghost in the Shell — Oshii, 1995*
- **Code Runner** `dark` — *Blade Runner 2049 — Villeneuve, 2017*
- **Chihiro's Code** `dark` — *Spirited Away — Miyazaki, 2001*
- **Codenonoké** `dark` — *Princess Mononoke — Miyazaki, 1997*
- **Code Out of Space** `dark` — *The Colour Out of Space — Lovecraft / Stanley, 2019*
- **Toxic Avencode** `dark` — *The Toxic Avenger — Kaufman, 1984*
- **Hannah Codeana** `light` — *Hannah Montana*
- **OLED** `dark`
